# Todo List App (Vanilla JavaScript)

A lightweight, responsive Todo List application built with pure JavaScript to practice Data Manipulation (CRUD), State Management, and browser persistent storage.

---

## Features

- **Full CRUD Operations:** Create, Read, Update, and Delete tasks.
- **Persistent Storage:** Uses browser `localStorage` to save tasks even after page refresh.
- **Dynamic Filtering:** Filter tasks by _All_, _Pending_, and _Completed_ statuses.
- **Responsive UI:** Clean design with smooth interactive states.

---

## Tech Stack & Concepts Applied

- **Frontend:** Vanilla JavaScript, HTML5, CSS3.
- **Data Handling:** Arrays & Objects manipulation (`.forEach()`, `.splice()`, `.push()`).
- **Storage:** JSON parsing (`JSON.stringify` / `JSON.parse`) with `localStorage`.
- **DOM Event Handling:** Event listeners for keyboard inputs, clicks, and dynamic state updates.
